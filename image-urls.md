# Image Assets

Static image assets for the geepers-chat UI.

## Locations

Place local image files in `client/public/images/`:

- `ai-avatar.png` — Avatar shown next to assistant messages
- `empty-state.png` — Illustration shown on the welcome/empty chat screen

These are referenced in:
- `client/src/components/ChatMessage.tsx` — `AI_AVATAR_URL`
- `client/src/components/EmptyState.tsx` — `EMPTY_STATE_IMG`
